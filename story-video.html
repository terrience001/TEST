📖 第一次使用？點這裡看安裝設定步驟（本機 / 雲端）

方式 A：完全本機（Mac 上跑，免費）

① 安裝 LM Studio（拆腳本）

  1. 到 lmstudio.ai 下載 Mac 版 .dmg，拖進「應用程式」
  2. 打開 LM Studio → 左邊搜尋 qwen2.5-7b-instruct → 下載 Q4_K_M 版本
  3. 左邊 Developer 分頁 → 載入模型 → 把 Status 切到 Running（位址 http://localhost:1234）

② 安裝 Draw Things（生圖）

  1. Mac App Store 搜尋 Draw Things，免費安裝
  2. 打開 App，左側會自動下載預設模型
  3. Settings → Advanced → Enable HTTP Server（Port 7860）

方式 B：雲端 GPU（更快，便宜租用）

  1. 到 Runpod / Vast.ai 租 RTX 4090（每小時約 NT$ 15–60）
  2. 選 AUTOMATIC1111 或 ComfyUI 模板開機
  3. 啟動時加參數 --cors-allow-origins=* 或 --api 開放跨網域
  4. 把 Runpod 給你的公開網址（如 https://xxx-7860.proxy.runpod.net）填進下面「圖像生成 API」
  5. LLM 同樣可以用雲端 OpenAI / Groq / OpenRouter（要 API key）

方式 C：付費 API（不用任何硬體）

直接用 OpenAI、Groq 等 API 拆腳本；圖片可以用 Replicate / fal.ai 跑 SDXL（要改下面 API 網址，並貼 API key）

⚠️ 注意：從 GitHub Pages（https）呼叫 localhost 多數瀏覽器會擋。如果不行，把這個 HTML 檔下載到 Mac 上，雙擊用 Chrome / Safari 直接開（file://），就不會被擋。
