📖 第一次使用？點這裡看完整安裝設定步驟

① 安裝 LM Studio（拆腳本用）

  1. 到 lmstudio.ai 下載 Mac 版 .dmg，拖進「應用程式」
  2. 打開 LM Studio → 左邊搜尋 qwen2.5-7b-instruct（或 llama-3.2-3b-instruct 比較輕）→ 下載 Q4_K_M 版本
  3. 左邊選 Developer（或 Local Server）分頁 → 上面載入剛下載的模型 → 把 Status 切成 Running
  4. 確認下方顯示 http://localhost:1234（這就是 API 位址，下面要填）

② 安裝 DrawThings（生圖用）

  1. Mac App Store 搜尋 Draw Things，免費安裝
  2. 打開 App，左側會自動下載預設模型（等它跑完，大約 2–4 GB）
  3. 選單列 Draw Things → Preferences / Settings → Advanced → 開啟 Enable HTTP Server（或叫 API Server），Port 預設 7860
  4. 確認 App 左下顯示 API 已啟動

③ 全部設定完後

回到本頁面按下方「檢查連線」按鈕，兩個綠燈都亮就可以開始貼故事了。
